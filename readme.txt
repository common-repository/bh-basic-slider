This is a Responsive basic slider Plugin

== Description ==

This plugin made with flexslider. Supported all crosebrowser and Responsive. very nice and helpful slider plugin

== Installation ==

1. Upload `BH Basic Slider` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `BH-Slider` in your page.
4. Place `<?php bh_slider_template();?>` in your templates
